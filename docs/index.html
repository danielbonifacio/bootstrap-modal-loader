Bootstrap Modal Loader

Loading personalizado dentro de modals, de fácil uso.

Modal title

Basta adicionar a classe 'modal-loader' dentro do modal que irá conter o loader.

O loader irá ocupar 100% da altura do que está dentro da div modal-body.

Este loader foi pensado para modals do bootstrap, mas, basta ver seu código fonte para ver como é simples sua aplicação em frameworks.
